較特殊的 <input> 型態


<input type="url">

<input type="date">

<input type="date" min="" max="">
能以此限制可選擇的日期區間
這個範例為 2021-06-23 至 2021-06-30

<input type="datetime-local">

能以此限制可選擇的日期與時間區間
這個範例為 2021-06-23 00:00 至 2021-06-30 23:59
<input type="datetime-local" min="" max="">

<input type="file">

<input type="file" multiple>

<input type="file" accept=".png, .jpg, .jpeg">
這個範例為指定 .png 或 .jpg 或 .jpeg 檔案類型

<input type="color">
