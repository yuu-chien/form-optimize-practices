通用屬性


<input type="text" required>
適用的 input 類型：text、search、url、tel、email、password、date pickers、number、checkbox、radio、file。

<input type="number" min="" max="">
適用的 input 類型：number、range、date、datetime、datetime-local、month、time、week

<input type="number" step="">
適用的 input 類型：number、range、date、datetime、datetime-local、month、time、week

<input type="text" minlength="" maxlength="">

<input type="text" readonly>
無法修改，使用者仍可用 tab 鍵切換並複製內容。

<input type="text" disabled>
無法使用與點擊的

<input type="text" inputmode="">
預設為 text，有 decimal, numeric, tel, search, email, url

<input type="text" inputmode="">
預設為 text，有 decimal, numeric, tel, search, email, url

<input type="email" multiple>
以逗號分隔，逗號後可加上空白
使用 multiple 時允許空值
適用的 input 類型：email、file

<input type="email" size="">
適用的 input 類型：text、search、tel、url、email、password

<input type="email" list="">
透過 list="" 指定相同 id 之 <datalist> 元素
