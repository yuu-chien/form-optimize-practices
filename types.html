type="email"

  • <input type="email">
  • 用於輸入電子信箱​
  • 預設之進行驗證的正則表達式：/^[a-zA-Z0-9.!#$%&'*+\/=?^_`{|}~-]+@[a-zA-Z0-9](?:[a-zA-Z0-9-]{0,61}​ [a-zA-Z0-9])?(?:\.[a-zA-Z0-9](?:[a-zA-Z0-9-]{0,61}[a-zA-Z0-9])?)*$/
  • 只能驗證輸入的資料是否符合格式，無法驗證其電子信箱是否存在

type="password"

  • <input type="password">
  • 用於輸入密碼​，輸入值會以隱碼呈現​

type="number"

  • <input type="number">
  • 輸入欄位右側會有上下兩個箭頭以供增加或減少數值，預設區間為 1​

type="tel"

  • <input type="tel">
  • 用於輸入電話號碼​​
  • 無預設的驗證格式，因為世界上的國家和地區電話號碼規則不盡相同
  • 與 number 主要差異於動態鍵盤的呈現​

type="url"

  • <input type="url">
  • 用於輸入網址​​
  • 驗證規則：http:// 或 https:// 開頭，並接至少一個字元
  • 只能驗證輸入的資料是否符合格式，無法驗證其電子信箱是否存在​

type="date"

  • <input type="date">
  • IE 11 不支援​
  • 預設情況下，不會驗證超過格式的輸入值​​
  • 根據瀏覽器設定的語言不同，顯示格式也會不一樣。例如繁體中文的格式為「年/月/日」，英文則為「mm/dd/yyyy」

type="datetime-local"

  • <input type="datetime-local">
  • Firefox 與 IE 11 不支援

type="file"

  • <input type="file">
  • HTMLInputElement.files 之屬性（name、lastModified、size、type）可用於得知文件名稱、大小、最後修改時間、MIME 類型
  • accept 屬性可用於指定上傳檔案格式

type="color"

  • <input type="color">
  • IE 11 不支援​
  • 不得為空值​​；預設色為 #000000​
  • ​只能輸入十六進制格式，無法處理帶透明度的色碼
