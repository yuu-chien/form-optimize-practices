通用屬性：required

  • <input type="text" required>
  • boolean​
  • 適用的 input 類型：text、search、url、tel、email、password、date pickers、number、checkbox、radio、file

通用屬性：readonly

  • <input type="text" value="try to copy" readonly>
  • boolean
  • 無法修改，使用者仍可用 tab 鍵切換並複製內容​​
  • 提交表單時會將值一起送出​​

通用屬性：disabled

  • <input type="text" value="try to copy" disabled>
  • boolean
  • 無法點選與修改​​

通用屬性：minlength、maxlength

  • <input type="text" minlength="6" maxlength="10">
  • number
  • IE 11 不支援 minlength
  • 適用的 input 類型：text、email、search、password、tel、url

通用屬性：min、max

  • <input type="number" min="-100" max="100">
  • string
  • 適用的 input 類型：number、range、date、datetime、datetime-local、month、time、week
<input type="date" min="2021-05-24" max="2021-06-23">
<input type="datetime-local" min="2021-06-23T00:00" max="2021-06-30T23:00">

通用屬性：step

  • <input type="number" step="10">
  • string
  • 適用的 input 類型：number、range、date、datetime、datetime-local、month、time、week
<input type="number" step="0.01">

通用屬性：multiple

  • boolean​
  • 以逗號分隔，逗號後可加上空白
  • 適用的 input 類型：email、file
<input type="email" multiple>
<input type="file" multiple>

通用屬性：size

  • <input type="text" size="6">
  • number
  • 若 type 類型為 passwordand、text，其以 em 為單位；其他則以 pixels 為單位
  • CSS 寬度權重高於此屬性
  • 適用的 input 類型：email、password、tel、text
  • 由於使用 CSS RESET，與實際尺寸不大符合。範例可參考 w3schools

通用屬性：pattern

  • <input type="text" pattern="(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{8,}" title="至少 8 個字元，並必須包括 1 個數字、大小寫字母各一個👀">
  • regexp
  • 適用的 input 類型：text、search、tel、url、email、password
  • 可將驗證規則寫在 title 中作為輔助提示
  • 主要用於增進 UX 體驗，驗證資料仍需交由伺服器端處理較安全
